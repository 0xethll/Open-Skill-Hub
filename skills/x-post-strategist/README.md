# X Post Strategist

**Platforms:** Claude · Openclaw · Codex
**Domain:** Writing

## Purpose

A strategy editor for writing X posts and threads. It turns rough ideas, short prompts, articles, reports, or technical material into clear, engaging, natural posts that fit the user's goal and account constraints.

## When to Use

- When you have an idea and want a stronger X post
- When you want to turn a long article or technical note into a thread
- When you need a sharper hook, clearer angle, or more natural voice
- When you want to know whether a post would benefit from an image
- When factual or sensitive claims need careful wording before posting

## What It Does

1. Clarifies the goal, audience, tone, account type, and format
2. Checks whether facts need verification before confident writing
3. Chooses a single post or thread based on the material
4. Writes strong hooks and publishable drafts
5. Removes generic AI tone while preserving the user's voice
6. Decides whether an image would improve the post
7. Provides an image brief when a visual would help
8. Scores and revises the draft for X fit, accuracy, quality, and human voice

## Notes

The skill defaults to English unless another language is requested. It is focused on X only, not LinkedIn, Threads, or general social media.

For real-time, financial, legal, medical, political, or other high-stakes topics, the agent should verify facts when possible and avoid presenting uncertainty as fact.
